SKT Mercury(머큐리) 공유기를 사용하면서 스마트폰의 **Windows App**(기존 RD Client)을 통해 PC에 접속하려면 크게 3가지 단계(PC 설정, 공유기 포트포워딩, 앱 연결)를 거쳐야 합니다.

---

### 1단계: 접속할 PC 설정 (Windows Pro 이상 필요)

기본적으로 Windows **Home** 버전은 원격 데스크톱 '호스트' 기능을 지원하지 않습니다. (Pro, Enterprise, Edition 등 필요)

1. **원격 데스크톱 활성화**: `설정` > `시스템` > `원격 데스크톱`에서 **[켬]**으로 변경합니다.
2. **전원 설정**: PC가 절전 모드로 들어가면 접속이 끊기므로, `설정` > `시스템` > `전원 및 취침`에서 **[사용 안 함]**으로 설정하세요.
3. **내부 IP 확인**: `Win + R` 키를 누르고 `cmd` 입력 후 `ipconfig`를 입력합니다. **IPv4 주소**(예: 192.168.45.XXX)를 메모해 두세요.

---

### 2단계: SKT Mercury 공유기 설정 (포트포워딩)

외부(LTE/5G)에서 공유기 내부의 PC를 찾아갈 수 있도록 길을 열어주는 작업입니다.

1. **관리자 페이지 접속**: 웹 브라우저 주소창에 `192.168.45.1`을 입력합니다.
2. **로그인**:

- **ID**: `admin`
- **PW**: `유선 MAC 주소 뒤 6자리_admin` (예: 공유기 스티커의 MAC 주소가 1234.56**ABC.DEF**라면 `ABCDEF_admin`)

3. **포트포워딩 설정**:

- 메뉴 위치: **고급 설정** > **NAT/라우터 설정** > **포트포워드**
- **내부 IP 주소**: 위에서 메모한 PC의 **IPv4 주소**
- **프로토콜**: TCP
- **포트 범위 (외부/내부)**: 모두 `3389` 입력 (보안을 위해 외부 포트는 다른 번호로 바꿔도 되지만, 기본값은 3389입니다.)
- [추가] 버튼 클릭 후 [적용]을 누릅니다.

---

### 3단계: 스마트폰 Windows App에서 접속

1. **외부 IP 확인**: 네이버에 '내 IP'를 검색하여 나오는 **공인 IP 주소**를 확인합니다.
2. **App 설정**:

- 앱 실행 후 `+` 아이콘 클릭 > **Add PC**
- **PC Name**: 확인한 **공인 IP 주소** 입력 (예: 211.xxx.xxx.xxx)
- **User Account**: PC 로그인 시 사용하는 계정(이메일 또는 ID)과 비밀번호 입력

3. **연결**: 저장된 항목을 터치하여 접속합니다.

---

노트북의 IP가 자꾸 변하면 공유기의 포트포워딩 설정(특정 IP로 길을 열어준 것)이 무용지물이 되기 때문에 **IP 고정은 원격 접속 설정에서 필수적인 과정**입니다.

방법은 크게 두 가지가 있는데, **공유기에서 설정하는 방법**이 가장 깔끔하고 오류가 적습니다.

---

### 방법 1: SKT 머큐리 공유기에서 고정 (추천)

공유기가 노트북의 고유 번호(MAC 주소)를 기억해서, 그 노트북에는 항상 똑같은 IP만 빌려주도록 설정하는 방식입니다.

1. **공유기 관리자 페이지 접속**: 브라우저에 `192.168.45.1` 입력 및 로그인.
2. **메뉴 이동**: **고급 설정** > **네트워크 관리** > **LAN 설정** (또는 **DHCP 고정 할당** 관련 메뉴).
3. **노트북 찾기**: 현재 연결된 기기 목록에서 본인의 노트북을 선택하거나, 직접 노트북의 **MAC 주소**와 **원하는 IP**(예: 192.168.45.20)를 입력합니다.

- _MAC 주소 확인법: PC에서 `cmd` 실행 후 `getmac` 입력_

4. **추가/적용**: [추가] 버튼을 누르고 반드시 **[설정 저장/적용]**을 클릭하세요.

---

### 방법 2: 노트북 윈도우 설정에서 고정

노트북 자체에서 "나는 무조건 이 IP만 쓸 거야"라고 지정하는 방식입니다.

1. `제어판` > `네트워크 및 공유 센터` > `어댑터 설정 변경`.
2. 현재 연결된 **Wi-Fi** 아이콘 마우스 우클릭 > **속성**.
3. **인터넷 프로토콜 버전 4(TCP/IPv4)** 선택 후 **속성** 클릭.
4. **[다음 IP 주소 사용]** 체크 후 아래 정보 입력:

- **IP 주소**: `192.168.45.20` (포트포워딩에 설정한 그 주소)
- **서브넷 마스크**: `255.255.255.0`
- **기본 게이트웨이**: `192.168.45.1`

5. **[다음 DNS 서버 주소 사용]** 체크:

- **기본 DNS**: `219.250.36.130` (SKT 기본 DNS)
- **보조 DNS**: `210.220.163.82`
